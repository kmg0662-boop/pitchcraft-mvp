# PitchCraft: 1호 Micro-SaaS 기획서

## 1. 개요
*   **서비스명**: PitchCraft (피치크래프트)
*   **목적**: 예비 창업자가 문장 2개만 입력하면, 엘리베이터 피치와 10장짜리 피치덱 목차를 자동 생성해주는 무료/프리미엄 하이브리드 웹 툴.
*   **비즈니스 목표**: 초기 사용자 트래픽 100명 달성 및 반응(이탈률, 체류 시간) 테스트. 반응이 좋으면 유료화 기능 추가, 없으면 과감히 폐기(Pivot).

## 2. 사용 기술 포지션
*   **프론트엔드 (에디)**: Vite + React + TypeScript + TailwindCSS(요청 시) 또는 순수 CSS (글래스모피즘, 고퀄리티 애니메이션 적용).
*   **비즈니스 로직 (코디/인터)**: 사용자 입력을 받아 Mock 데이터를 내려주거나, 추후 실제 AI API(Claude 등)와 연동할 수 있는 구조.
*   **카피라이팅 (라익/페이퍼로지)**: 피치덱 목차와 카피의 퀄리티 로직 설계.

## 3. 핵심 UI 흐름
1.  **메인 랜딩**: 강렬하고 유려한 타이포그래피. "당신의 아이디어를 5초 만에 피치덱으로."
2.  **입력창**: "어떤 문제를 어떻게 해결하시나요?" (1-2문장 입력)
3.  **로딩 애니메이션**: 요원들이 작업하는 듯한 세련된 모션.
4.  **결과 화면**: 
    1. 엘리베이터 피치 1문단
    2. 피치덱 10 Slide 구조 (마크다운 테이블 또는 가이드 형태)
5.  **CTA**: "PDF로 내보내기 (이메일 구독 시 무료)" -> 잠재 고객 리드 확보.
